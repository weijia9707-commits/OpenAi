FAQ

HEX is a hexadecimal color code (e.g., #FF0000), while RGB represents red, green, blue values (e.g., 255, 0, 0). They are essentially the same, just different representations. HEX is more concise and commonly used in web CSS; RGB is more intuitive for understanding color composition.

Complementary colors are two colors opposite each other on the color wheel, such as red and cyan, or blue and orange. When used together, they create strong contrast and visual impact, often used in designs that need to highlight key elements.

HSL represents Hue, Saturation, and Lightness. Compared to RGB, HSL is more aligned with human perception of color, allowing more intuitive adjustments to color brightness and vibrancy.

Click the favorite button next to a color to save it to browser local storage. Saved colors will automatically load on your next visit, with a maximum of 16 favorite colors.

CMYK is a four-color printing mode representing Cyan, Magenta, Yellow, and Key (Black). It is mainly used for print design and differs from the RGB mode used for screen display.

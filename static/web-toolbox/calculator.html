❓ FAQ

What functions does the scientific calculator support? ▼
Supports sin, cos, tan, log, ln, factorial, power, square root and more. Keyboard input supported: Enter to calculate, Escape to clear.
What types of unit conversion are supported? ▼
Supports 16 types: length, area, volume, temperature, time, speed, pressure, power, energy, angle, data size, density, force. Each includes multiple common units.
How to use the subnet mask calculator? ▼
Enter IP address and subnet mask or CIDR (e.g., /24). The tool calculates network address, broadcast address, host range, host count, wildcard mask, and binary IP.
How to use the interest calculator? ▼
Enter principal, rate and term, select repayment method. The tool calculates monthly payment, total payment, and total interest. Equal principal also shows first/last month payment and monthly decrease.
What is the calculation precision? ▼
Uses JavaScript floating-point arithmetic with precision of about 15-17 significant digits. Unit conversion results automatically remove trailing zeros.

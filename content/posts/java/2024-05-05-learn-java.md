#### 1.2 变量

**定义：** 变量是用来记录程序中的数据的。其本质上是内存中的一块区域，你可以把这块区域理解成一个小盒子，盒子里放的东西就是变量记录的数据。

**格式：** `数据类型 变量名 = 初始值;`

**作用：** 使用变量来记录数据，对于数据的管理更为灵活。
